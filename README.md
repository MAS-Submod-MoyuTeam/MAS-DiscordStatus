## 目前支持的功能 

- 显示所在的房间
- 显示小游戏NOU和上吊小人
- 兼容子模组“网易云音乐”和“Extra Plus”
- 显示在线时长
- 当你在开发时，显示你在摸鱼
- 在不同的房间显示不同的状态图
